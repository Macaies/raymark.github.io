Racing QLD Sectionals – Data Pipeline & Analytics

End-to-end R pipeline (RStudio + Shiny) that downloads Racing QLD sectional data, cleans and normalises race/horse features, and builds a historical dataset for modelling.

  • Automated ZIP/XML download → cleaning → feature engineering → combined CSV.
  • Shiny UI (in progress) with date filters, ETL “Run” button, and dataset preview/export.
